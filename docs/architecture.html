2. Star Schema (Dimensional Model)

Classic dimensional modeling: one fact table, three dimensions. Measures live on the fact; slowly-changing attributes live on the dims.

4. Request Flow (Ask AI — Claude text-to-SQL)

User question → sent to Claude with full schema context → safety filter → SQL executes on SQLite → Claude summarizes the result in English.
